### 视觉风格（Numra 设计语言）

- **主色调**: 柔和的薰衣草紫 (#7C6FF7 / #8B83BA) 作为品牌色
- **辅助色**: 薄荷绿 (#4ECDC4 / #6BCB77) 用于健康/达成类指标
- **背景**: 暖白基底 (#F8F7FC / #F5F3FA)，替代当前的深黑
- **卡片**: 纯白 (#FFFFFF)，圆角 16px，轻柔阴影 (0 2px 12px rgba(124,111,247,0.08))
- **文字**: 深灰 (#2D2B3A) 主文字，中灰 (#7C7A8C) 辅助文字
- **边框**: 淡紫灰 (#E8E6F0)，只在必要时出现
- **圆角**: 全局 12-16px 大圆角，按钮 8-10px
- **阴影**: 柔和低饱和度投影，替代硬边框分隔
- **图标**: 线性圆角图标风格 (lucide-react 现成可用)
- **数据可视化**: 柔和渐变柱状图、环形进度、平滑折线

### 看板内容（WellTrack 四支柱 + AI 洞察）

#### 第一行：三大核心指标卡（1:1:1 等宽）

1. **今日训练卡**
   - 训练状态图标 + 文字（已完成/待训练/休息日）
   - 焦点肌群标签
   - 今日训练建议一句话
   - 状态指示色：绿=已完成，琥珀=待训练，灰=休息日

2. **恢复状态卡**
   - 睡眠时长数字 + 单位（大号字体）
   - 三个微型指标：疲劳度/疼痛/压力（小号横向排列）
   - 每项用 1-10 数字 + 进度条色（绿→黄→红）
   - 睡眠质量标签（优秀/良好/需改善）

3. **营养概览卡**
   - 热量环形进度：当前/目标
   - 三大营养素横向条形：蛋白质/碳水/脂肪
   - 饮水进度条
   - 颜色编码：蛋白质=品牌紫，碳水=薄荷绿，脂肪=暖琥珀

#### 第二行：趋势图表（1:1 等宽）

4. **体重趋势图**
   - 7 天平滑折线图（recharts）
   - 显示起始/当前体重数值
   - 趋势方向箭头（↑/↓/→）+ 变化量
   - 淡紫色渐变填充区域

5. **目标进度卡**
   - 大型环形进度图（中间显示百分比或剩余天数）
   - 下方：当前体重 → 目标体重，差距 kg
   - 进度标签（健康/过慢/超额/数据不足）
   - 语义色：健康=绿，过慢=琥珀，超额=红

#### 第三行：全宽 AI 洞察

6. **AI 个性化洞察卡**
   - 卡片顶部左侧 AI 图标 + "AI Coach 洞察"
   - 基于后台数据的自动分析文本（2-4 条要点）
   - 例如："本周平均睡眠 6.2h 低于 7h 目标，建议今晚提前 30 分钟入睡"
   - 例如："深蹲训练量较上周增长 15%，恢复指标正常，继续保持"
   - 淡紫色左边框，浅紫背景

### 与当前的主要变化

| 维度   | 当前              | 新设计                   |
| ------ | ----------------- | ------------------------ |
| 配色   | 深黑+蓝色强调     | 暖白基底+薰衣草紫+薄荷绿 |
| 卡片   | 深色背景+硬边框   | 白色卡片+柔和阴影+大圆角 |
| 布局   | 2x2 网格+底部图表 | 3-2-1 行式网格           |
| 训练卡 | 文字+状态         | 图标+标签+建议           |
| 恢复卡 | 4项纵向排列       | 睡眠大字+3微型横向       |
| 营养卡 | 5项纵向排列       | 热量环+宏观条形+饮水     |
| 体重图 | div柱状图         | recharts 折线图          |
| 目标   | 条形进度条        | 环形进度图               |
| 新增   | -                 | AI 洞察卡                |
| 状态栏 | 横跨底部          | 保留，适配新配色         |

## 实施步骤

1. **globals.css** — 全面重写 CSS 变量和基础样式（Numra 配色系统）
2. **DashboardCards.tsx** — 重写所有卡片组件（WellTrack 内容组织）
3. **DashboardPanel.tsx** — 添加头部 + 时间选择器，重组网格布局
4. **App.tsx** — 调整左右分栏比例（Cloudbrand 框架感）
5. **StatusBar.tsx** — 适配新配色
6. **ChatPanel / MessageList** — 适配新配色（卡片白底+阴影替代暗色）
7. **TypeScript + Vite 验证**
